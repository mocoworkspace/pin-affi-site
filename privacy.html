プライバシーポリシー

最終更新日: 2026年6月23日

本プライバシーポリシーは、mocolab（以下「運営者」）が提供する個人用ツール 「pin-affi」（以下「本アプリ」）における、情報の取り扱いについて定めるものです。 本アプリは運営者本人が自身の Pinterest アカウントを運用するための個人用ツールであり、 一般向けに公開・販売するものではありません。

1. 取得する情報

  • アカウント情報: ログインに用いるメールアドレス。
  • コンテンツ情報: 利用者が作成したピンのタイトル・説明・画像・ ボード情報・アフィリエイトリンク・予約日時など。
  • Pinterest 連携情報: Pinterest 公式 API を利用するための アクセストークン／リフレッシュトークン、および連携アカウントの識別子。

2. 利用目的

  • 本アプリの認証およびデータの保存・表示のため。
  • 利用者本人の指示に基づき、Pinterest 公式 API（v5）を通じて ピンの作成・予約投稿を行うため。
  • 投稿状況の記録、エラーの把握、機能改善のため。

3. Pinterest データの取り扱い

本アプリは Pinterest API を利用し、利用者本人が連携を許可した範囲（ボードの参照、 ピンの作成・参照、アカウント情報の参照）でのみ Pinterest のデータにアクセスします。 取得した情報は本アプリの機能提供の目的にのみ使用し、 第三者への提供・販売・広告目的での利用は行いません。 本アプリの利用は、Pinterest の 開発者向け規約 に従います。

4. データの保存と保護

  • データは Supabase（PostgreSQL）上に保存されます。
  • 各データは行レベルセキュリティ（RLS）により、本人のみがアクセスできるよう制御しています。
  • Pinterest のアクセストークン等の認証情報は暗号化して保存し、 クライアント（アプリ）からは参照できない構成としています。

5. 第三者サービス

本アプリは以下の外部サービスを利用します。各サービスのプライバシーポリシーが適用されます。

  • Pinterest（コンテンツの投稿・参照）
  • Supabase（認証・データベース・ストレージ）

6. データの削除

利用者はアプリ内でピン等のデータを削除できます。Pinterest 連携の解除や アカウントの削除を希望する場合は、下記の連絡先までご連絡ください。 連携解除後、保存されたトークンは無効化・削除されます。

7. 改定

本ポリシーは必要に応じて改定されることがあります。改定時は本ページにて更新します。
